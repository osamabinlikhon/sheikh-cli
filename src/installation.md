# ইনস্টলেশন গাইড

## পূর্বশর্ত

শেখ CLI টুলটি ইনস্টল করার আগে নিশ্চিত হন যে আপনার সিস্টেমে পাইথন ৩.৯ বা তার উপরের সংস্করণ ইনস্টল করা আছে। পাইথন ইনস্টল করা থাকলে টার্মিনালে `python --version` কমান্ড চালিয়ে যাচাই করুন।

## পাইথন ডিপেন্ডেন্সি ইনস্টলেশন

প্রথমে প্রজেক্ট রিপোজিটরি ক্লোন করুন এবং প্রয়োজনীয় প্যাকেজ ইনস্টল করুন। নিচের কমান্ডগুলো টার্মিনালে চালান:

```bash
git clone https://github.com/osamabinlikhon/sheikh-cli.git
cd sheikh-cli
pip install -r requirements.txt
```

এই কমান্ডগুলো সমস্ত প্রয়োজনীয় পাইথন প্যাকেজ ইনস্টল করবে যা টুলটি চালাতে প্রয়োজন।

## Playwright ইনস্টলেশন (শক্তিশালী স্ক্রেপারের জন্য)

শেখ পাওয়ার স্ক্রেপার ব্যবহার করতে চাইলে Playwright ইনস্টল করতে হবে। পাইথন ভার্সনের জন্য নিচের কমান্ড চালান:

```bash
pip install playwright
playwright install
```

Node.js ভার্সনের জন্য npm প্যাকেজ ম্যানেজার ব্যবহার করুন:

```bash
npm install
npx playwright install
```

## ভার্চুয়াল এনভায়রনমেন্ট সেটআপ (সুপারিশকৃত)

প্রজেক্টের নির্ভরতা সিস্টেমের অন্যান্য প্যাকেজ থেকে আলাদা রাখতে ভার্চুয়াল এনভায়রনমেন্ট ব্যবহার করা সুপারিশকৃত। নিচের ধাপগুলো অনুসরণ করুন:

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac এর জন্য
# অথবা Windows এর জন্য: venv\Scripts\activate
pip install -r requirements.txt
```

## যাচাইকরণ

ইনস্টলেশন সফল হয়েছে কিনা যাচাই করতে টুলটির ভার্সন চেক করুন:

```bash
python sheikh --version
```

সফল হলে টুলটির সংস্করণ নম্বর দেখাবে।
