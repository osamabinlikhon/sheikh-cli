# কমান্ড রেফারেন্স

## স্ক্রেপ কমান্ড

স্ক্রেপ কমান্ডটি বাংলা ওয়েবসাইট থেকে ডেটা সংগ্রহ করতে ব্যবহৃত হয়।

### সিনট্যাক্স

```bash
sheikh scrape <টার্গেট> [max_urls] [অপশন]
```

### আর্গুমেন্টস

প্রথম আর্গুমেন্ট হল টার্গেট যা স্ক্রেপ করার ডোমেইন বা URL। দ্বিতীয় আর্গুমেন্ট হল max_urls যা সর্বোচ্চ স্ক্রেপ করার URL সংখ্যা, ডিফল্ট মান ১০০।

### অপশনসমূহ

`-o, --output` অপশন দিয়ে আউটপুট ফাইল পাথ নির্ধারণ করা যায়। `-w, --workers` অপশন দিয়ে কনকারেন্ট কর্মী সংখ্যা নির্ধারণ করা যায়, ডিফল্ট মান ১০। `-r, --rate` অপশন দিয়ে প্রতি সেকেন্ডে রিকোয়েস্ট সংখ্যা নির্ধারণ করা যায়, ডিফল্ট মান ০.৫। `--resume` অপশন দিয়ে আগের সেশন থেকে চালিয়ে যাওয়া যায়। `--no-robots` অপশন দিয়ে robots.txt চেক বাইপাস করা যায়। `-m, --mode` অপশন দিয়ে ডিসকভারি মোড নির্ধারণ করা যায়, বিকল্প: domain বা hub। `-d, --allow-domains` অপশন দিয়ে হাব মোডে অনুমোদিত ডোমেইন ফিল্টার করা যায়।

## সিনট্যাক্স কমান্ড

সিনট্যাক্স কমান্ডটি ডেটাসেট ফাইলের স্কিমা এবং বৈধতা যাচাই করতে ব্যবহৃত হয়।

### সিনট্যাক্স

```bash
sheikh syntax <ফাইল>
```

### ব্যবহারের উদাহরণ

```bash
sheikh syntax data/train.jsonl
```

এই কমান্ডটি `data/train.jsonl` ফাইলের সিনট্যাক্স যাচাই করবে এবং কোনো ত্রুটি পাওয়া গেলে দেখাবে।

## স্ট্যাটস কমান্ড

স্ট্যাটস কমান্ডটি ডেটাসেটের গুণমান এবং দৈর্ঘ্য পরিসংখ্যান গণনা করতে ব্যবহৃত হয়।

### সিনট্যাক্স

```bash
sheikh stats <ফাইল>
```

### ব্যবহারের উদাহরণ

```bash
sheikh stats data/train.jsonl
```

এই কমান্ডটি ফাইলের মোট স্যাম্পল, ক্যারেক্টার এবং শব্দ গণনা এবং গড়, মিডিয়ান এবং স্ট্যান্ডার্ড ডেভিয়েশন প্রদর্শন করবে।

## স্প্লিট কমান্ড

স্প্লিট কমান্ডটি ডেটাসেটকে ট্রেন এবং ভ্যালিডেশন সেটে ভাগ করতে ব্যবহৃত হয়।

### সিনট্যাক্স

```bash
sheikh split <ফাইল> [ratio] [অপশন]
```

### ব্যবহারের উদাহরণ

```bash
sheikh split data/train.jsonl 0.8
```

এই কমান্ডটি ডেটাসেটকে ৮০% ট্রেন এবং ২০% ভ্যালিডেশনে ভাগ করবে। `--no-stratify` অপশন দিয়ে স্ট্র্যাটিফাইড স্প্লিট বন্ধ করা যায়।

## পুশ কমান্ড

পুশ কমান্ডটি ডেটাসেট Hugging Face Hub এ আপলোড করতে ব্যবহৃত হয়।

### সিনট্যাক্স

```bash
sheikh push <ফাইল> [অপশন]
```

### ব্যবহারের উদাহরণ

```bash
sheikh push data/train.jsonl --repo-id osamabinlikhon/bangla-dataset
```

`-r, --repo-id` অপশন দিয়ে কাস্টম রিপোজিটরি ID নির্ধারণ করা যায়। `--private` অপশন দিয়ে প্রাইভেট রিপোজিটরি তৈরি করা যায়।
