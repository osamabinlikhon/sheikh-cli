# ব্যবহার নির্দেশিকা

## মূল কমান্ড সিনট্যাক্স

শেখ CLI টুলের মূল কমান্ড সিনট্যাক্স নিম্নরূপ:

```bash
sheikh <কমান্ড> [অপশন] [আর্গুমেন্ট]
```

প্রতিটি কমান্ডের নিজস্ব সেট অফ অপশন এবং আর্গুমেন্ট রয়েছে যা কমান্ডের কার্যকারিতা নির্ধারণ করে।

## স্ক্রেপিং মোড

স্ক্রেপিং কাজ সম্পাদনের জন্য দুটি মোড উপলব্ধ রয়েছে। প্রথম মোড হল ডোমেইন মোড যা সাইটম্যাপ থেকে URL বের করে এবং এটি ডিফল্ট মোড। দ্বিতীয় মোড হল হাব মোড যা হাব পেজ থেকে সরাসরি লিংক বের করে।

### ডোমেইন মোড ব্যবহার

ডোমেইন মোডে স্ক্রেপিং করতে নিচের কমান্ড ব্যবহার করুন:

```bash
sheikh scrape allbanglachoti.com 1000
```

এই কমান্ডটি `allbanglachoti.com` ডোমেইন থেকে সর্বোচ্চ ১০০০টি URL স্ক্রেপ করবে।

### হাব মোড ব্যবহার

হাব মোডে স্ক্রেপিং করতে `--mode hub` অপশন ব্যবহার করুন:

```bash
sheikh scrape https://sites.google.com/view/allbanglachoti/bangla-choti 500 --mode hub
```

এই কমান্ডটি প্রদত্ত গুগল সাইট পেজ থেকে সর্বোচ্চ ৫০০টি লিংক বের করবে এবং স্ক্রেপ করবে।

## উন্নত অপশনসমূহ

### কর্মী সংখ্যা নির্ধারণ

একাধিক কর্মী ব্যবহার করে স্ক্রেপিং গতি বাড়ানো যায়। `--workers` অপশন দিয়ে কর্মী সংখ্যা নির্ধারণ করুন:

```bash
sheikh scrape allbanglachoti.com 1000 --workers 20
```

### রেট লিমিটিং

সার্ভার ব্লক করা এড়াতে রেট লিমিট সেট করুন:

```bash
sheikh scrape allbanglachoti.com 1000 --rate 0.5
```

এই কমান্ডটি প্রতি কর্মীর জন্য প্রতি ০.৫ সেকেন্ডে একটি রিকোয়েস্ট পাঠাবে।

### রিজিউম ফাংশনালিটি

যদি স্ক্রেপিং মাঝখানে বন্ধ হয়ে যায়, `--resume` অপশন দিয়ে চালিয়ে যেতে পারবেন:

```bash
sheikh scrape allbanglachoti.com 1000 --resume
```

## আউটপুট ফাইল পাথ

ডিফল্টভাবে আউটপুট `data/raw` ফোল্ডারে সংরক্ষিত হয়। কাস্টম পাথ নির্ধারণ করতে `--output` অপশন ব্যবহার করুন:

```bash
sheikh scrape allbanglachoti.com 1000 --output my_custom_data.csv
```
