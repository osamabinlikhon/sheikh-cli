# OPENCODE.md - OpenCode AI Guidelines

## Project Context

Sheikh CLI - Agentic coding tool for Termux and terminal environments.

## Quick Commands

```bash
npm run dev    # Development with hot reload
npm run build  # Build with esbuild
npm run start  # Run compiled
npx tsc --noEmit  # Type check
```

## Tech Stack

- TypeScript + React (Ink)
- esbuild for bundling
- MCP protocol

## Files

- `src/index.tsx` - Entry point
- `dist/index.js` - Built output
- `package.json` - Package config

## Notes

- Package name: `sheikh-termux`
- TypeScript strict mode: disabled
- Editor: nano preferred
